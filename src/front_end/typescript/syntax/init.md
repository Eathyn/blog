# Init

## 安装

> Reference: [Download TypeScript](https://www.typescriptlang.org/download)

- 本地安装：

```bash
npm install typescript --save-dev
```

- 全局安装：

```bash
npm install -g typescript
```

## 生成 tsconfig.json

```bash
tsc --init
```

## 运行

```bash
npx tsc <ts file>
```
