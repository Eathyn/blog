# Commands

## init

- `npm init` 生成 `package.json` 文件
- 别名: `create`

## install

- `npm install`：安装所有包
- `npm install <package>`：安装某一个包

### 描述

- `--global` / `-g`：全局安装
- `--save-dev` / `-D`：开发的时候需要的依赖包，出现在 `devDependencies`。

## uninstall

- `npm uninstall <package>`：卸载包
