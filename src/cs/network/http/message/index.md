# HTTP Message

- HTTP 报文（message）有两种类型：请求（request）和响应（response）。

## Request

- 请求的组成：
  - HTTP method
  - resource path
  - version of the protocol
  - headers
  - body

![request](./images/request.png =x250)

## Response

- 响应的组成：
  - version of the protocol
  - status code
  - status message
  - headers
  - body

![response](./images/response.png =x320)
