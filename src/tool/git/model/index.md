# Model

## Tracked and Untracked File

> Reference: 
> - Pro Git: page-27

- tracked file: 使用 `git add` 添加过的文件。
- untracked file: 未使用 `git add` 添加过的文件。

## The Three States

> Reference: 
> - Pro Git: page-15

- Git 中的文件有三种状态：modified（已修改） / staged（已暂存） / committed（已提交）
