# 经营自己的职业生涯就像经营一家企业

## 拥有商业心态

- 把自己当作一家企业去思考，把雇主当作自己软件开发企业的一个客户

## 将思维模式切换为“经营企业”

- 软件开发人员售卖的就是他们把一个想法变成一个数字化的现实产品的能力
- 作为一名软件开发人员，你提供的服务具备有形价值，你要传达的不仅是这款软件的价值是什么，还要让它与成千上万别的软件开发人员提供的服务有何不同
- 产品营销做得越好，你就能给服务定越高的价格，也越有机会吸引更多的潜在客户
- 规划职业生涯
  - 专注于我正在提供的服务，并营销这项服务
  - 提升我的服务
  - 思考我可以专注为哪一特定类型的客户或行业提供特定的服务
  - 集中精力成为一位专家，专门为某一特定类型的客户提供专业的整体服务
- 想想如何更好地宣传你的服务，如何更好地找到你的客户
