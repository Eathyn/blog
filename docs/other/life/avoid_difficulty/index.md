# Avoid Difficulty

## Phenomenon

- 学习时遇到困难就逃避，然后去做其他事情

## Analysis

- 有完美主义的思想，太高估自己，觉得在学习的过程中不会遇到不懂的问题，只需要看文字就能理解
- 因为完美主义的存在，不能接受自己不懂，内心会逐渐烦躁和失望，学习的注意力也会下降，然后就放弃继续解决问题去做别的事情了
- 适应不良的完美主义追求避免犯错，思维方式是：如果我完美地完成了这件事情，那么我能避免自我羞愧感、指责和他人的指指点点

### 适应不良的完美主义通常有以下几点表现

- 拖延：由于恐惧失败和犯错，完美主义者会尽可能地拖延事情的开始，如果中途遇到困难，他们也会很容易就放弃，内心觉得只要逃避掉这个困难就不需要面对自己无法解决这个困难的现实
- 抵触负面意见：对他人的意见或批评特别敏感和反感，害怕别人知道自己犯了错

### 为什么会变成完美主义者

- 害怕失败：完美主义者把失败归结于自身能力不足，从而得承认自己的无能
- 全或无的想法：完美主义者缺乏折中的思想，只要存在一点与计划偏差的地方就会全盘否定整个计划
- 幼年缺乏安全感：完美的成绩是为了讨好父母而不是基于自己的兴趣爱好，长期以往导致了做任何事都是外部驱动而不是自我驱动，但是外部驱动无法给自己足够的耐心解决问题

### 完美主义者为了完美付出的代价

- 被负面情绪困扰：完美主义者会因为一点偏差放弃整天的任务，放弃后又会产生自责感，之后又会影响到下一个任务的开始
- 亲密关系不良：完美主义者对自己要求高的同时也对其他人要求高；避免向亲友暴露自己的真实情感和不安，阻止了亲密关系的发展

### 如何缓解适应不良的完美主义

- 建立计划时要预留出充分的时间冷静地分析可行性，实际计划要比理想的计划略微降低，以确保大部分内容都可以完成，建立正向反馈
- 日常学习要接受并容纳困难，告诫自己只是普通人，不要气馁，要以自我驱动的思想慢慢解决困难
- 一天中的某个小任务没有完成时不要气馁，更不要放弃后续任务的执行。对没有完成的任务进行分析，发现原因然后调整
