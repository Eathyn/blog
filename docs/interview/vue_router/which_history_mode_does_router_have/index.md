# Which History Mode does Router Have

## Hash Mode

- Hash mode has `#` symbol.
- Hash mode doesn't support History API
- The Server doesn't need configuration.

## History Mode

- History mode doesn't have `#` symbol.
- Hash mode supports History API
- The Server needs configuration.

## Memory Mode

- Memory mode is perfect for node environment and SSR.
