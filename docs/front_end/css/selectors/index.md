# Selectors

## Universal Selector

```html
<div>hello</div>
```

```css
* {
	color: red;
}
```

## Type Selector

```html
<div>hello</div>
```

```css
div {
	color: red;
}
```

## Class Selector

```html
<div class="item">hello</div>
```

```css
.item {
	color: red;
}
```

## ID Selector

```html
<div id="item">hello</div>
```

```css
#item {
	color: red;
}
```

## Attribute Selector

```html
<div id="item">hello</div>
```

```css
[id] {
	color: red;
}
```
