# 助动词

## 58 表示能力 "会" 的 can

- 表示会... 时用 `can`

```markdown
I can swim.
```

- 加 `not` 形成否定句

```markdown
I can't swim.
```

- `can` / `can not` 后面需要有动词，且动词为原形

## 59 询问能力的 "会...吗"

```markdown
Can you swim?
```

- 回答为 `Yes, ... can` / `No, ... can't`

```markdown
Can you swim? -> Yes, I can. / No, I can't.
```

- 在使用 `can` 的句子中，动词使用原形

```markdown
Can he swim?
```

- `can not` = `cannot` = `can't`

## 60 "我可以...吗" "你可以...吗"

- `Can I` 可以用在表示请求他人同意
- 回答他人请求时，不是用 `Yes, you can` / `No, you can't` 回答，而是使用 `sure` / `all right` / `ok` 表示同意，或者使用 `I'm sorry + 理由` 表示拒绝

```markdown
Can I use your pen? -> Sure. / I'm sorry, it's not working.
```

- `can you...` 可以用在询问 "你能...吗" 或请求他人帮忙

```markdown
Can you swim? 你会游泳吗?
Can you open the door? 你可以帮我开一下门吗?
```

- `Can I ...?` 不如 `May I ...?` 客气，但是比较轻松、亲切，后面加 `please` 就比较得体

```markdown
Can I use your pen please?
```
