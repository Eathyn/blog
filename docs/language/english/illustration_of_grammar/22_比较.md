# 比较

## 82 比...更...

- 比较级: 词尾 + er

```markdown
tall -> taller
```

- than... : 比...

```markdown
Tom is taller than Ken.
```

- 比较 A 和 B

```markdown
Which is taller, Tom or Ken?
```

## 83 ...之中最...

- 最高级: 词尾 + est

```markdown
Tom is the tallest.
```

- 最高级后 + of/in
  - 接复数词语就用 of
  - 接地点、团体就用 in

```markdown
Tom is tallest in the three.
Tom is tallest in China.
```

- 形容词最高级前面要加 the，副词最高级则不需要

## 84 要注意的比较级变化-1

- 单词结尾为 e，则加 r / st

```markdown
large -> larger -> largest
```

- a, e, i, o, u 以外 + y 结尾的单词，则变 y 为 i 再加 er / est

```markdown
busy -> busier -> busiest
```

- 词尾为辅音+重读的元音+辅音的单词，双写最后一个字母再加 er / est

```markdown
big -> bigger -> biggest
```

- 不规则变化

```markdown
good / well-> better -> best
many / much -> more -> most
bad -> worse -> worst
little -> less -> least
```

## 85 要注意的比较级变化 - 2

- 三音节以上的单词和词尾是 ful, ous, ing 的单词的比较级或最高级不加 er/est，而是加 more/most

```markdown
popular -> more popular -> most popular
```

## 86 和...一样

- as...as: 同...一样

```markdown
as big as
```

- not as... as...: 没有和...一样...

```markdown
I am not as tall as Tom.
```

- 第一个 as 表示程度相同地的副词，第二个 as 表示和...比起来的连接词

- as...as... 也可以表达几倍

```markdown
A is three times as large as B. A 是 B 的三倍大
```

## 87 比较句型的总结

- 比较级 + than any other + 单数名词: 比其他任何...都

```markdown
Tom is taller than any other student in the class.
```

## 复习测验

- like A better than B: 比起B更喜欢A

```markdown
I like computer science better than economics.
```

- like ... (the) best: 最喜欢...

```markdown
I like computer science (the) best of all subject.
What sport do you like (the) best?
```
