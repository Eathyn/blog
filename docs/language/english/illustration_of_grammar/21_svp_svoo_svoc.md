# SVP, SVOO, SVOC

## 78 "成为..." "看起来..."

- 表示 "成为..." 可以使用 `become`，后面接名词或形容词

```markdown
I want to become a singer.
I want to become famous.
```

- 表示 "看起来..." 可以使用 `look`，后面接形容词

```markdown
He looks happy.
```

- 表示 "听起来" 可以使用 `sound`，后面接形容词

```markdown
It sounds interesting.
```

- 表示 "变得" 可以使用 `get`，后面接形容词

```markdown
She got angry.
```

- SVP：S(subject 主语), V(verb 动词), P(predicative 表语)

```markdown
I am happy. (s: I, v: am, p: happy)
He looks happy. (s: He, v: looks, p: happy)
```

- `look(s) like`: 看起来像

## 79 "给..." "让某人看..."

- 给: `give`

```markdown
I gave him a present.
```

- 看: `show`

```markdown
Show me your dog.
```

- `give/show/tell + 人 + 物` 的句型中有两个宾语，称为 `SVOO` 句型

```markdown
I gave him a present. // 直接宾语: a present | 间接宾语: him
```

## 80 "称呼A为B" "使A变成B"

- 称呼A为B: call A B

```markdown
We call him Tom.
```

- 使A变成B: make A B

```markdown
This song makes me happy.
```

- "主语+动词+宾语+补语" 称为 "SVOC" 句型

```markdown
// Subject (主语): this song
// Verb (动词): make
// Object (宾语): me
// Complement (补语): happy -> 对宾语的补充
This song makes me happy.
```

## 复习测验

- 使用介词 `to` 可以改变 "人" 和 "物" 的顺序

```markdown
I sent him a present. -> I sent a present to him.
```
