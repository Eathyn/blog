# 打好基础，精益求精 - 6

- S (subject): 主语
- V (verb): 动词
- P (predicative): 表语 (说明主语的名词或形容词)
- O (object): 宾语
- C (complement): 补语 (对宾语进行补充说明)

## SV

- 动词后面可以加修饰词 (副词, 或者以介词引导的短语)
- 不接宾语的动词称为不及物动词

```markdown
She sings very well.

- S: she
- V: sing
- 修饰词: very well
```

## SVP

- 表语: 说明主语的名词或形容词

```markdown
I am busy.

- S: I
- V: am
- P: busy
```

## SVO

- 后面需要接宾语的动词称为及物动词

```markdown
I play tennis.

- S: I
- V: play
- O: tennis
```

## SVOO

- 第一个宾语称为间接宾语，第二个宾语称为直接宾语

```markdown
He gave me a book

- S: he
- V: give
- O: me
- O: a book
```

## SVOC

- 补语: 对宾语进行补充说明

```markdown
We call him Ken.

- S: we
- V: call
- O: him
- C: Ken
```
