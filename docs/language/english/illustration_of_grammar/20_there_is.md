# There is ...

## 75 "有..."

- 复数用 "there are", 单数用 "there is"

```markdown
There is a desk.
There are two desks.
```

- 表达过去式时 be 动词要用过去式

```markdown
There was a desk.
```

- `There be` 后面不会接有 `the`, `my`, `your` 的名词

```markdown
There is your bag on the desk. // error
Your bag is on the desk. // right
```

## 76 "没有..."

- 否定句里经常使用 `any`

```markdown
There are not any books.
```

- not any + n. = no + n.

```markdown
There are no books.
```

- 改为否定句

```markdown
There are some flowers. -> There are not any flowers.
```

## 77 有...吗

- `are there` 中经常使用 `any`

```markdown
Is there a hospital near here?
Ara there any comics in the library?
```

- 使用 `yes/no, there be` 回答

```markdown
Is there a hospital near here?
Yes, there is. / No, there isn't.
```

- 回答也要遵守时态

```markdown
Was there a hospital here?
Yes, there was. / No, there wasn't.
```

- `there will be` 表示未来有...

```markdown
There will be an exam next week.
```
