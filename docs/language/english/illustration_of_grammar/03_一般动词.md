# 一般动词

## 07 各式各样的动词

- 动词分为 be 动词 / 一般动词 / 助动词 / 情态动词

- 一般动词例子 : play / like / have / study

- 如果要用一般动词就不需要 be 动词

```markdown
Error:
I am like music.

Right:
I like music.
```

## 08 第三人称是什么

- 人称有第一人称、第二人称和第三人称三种

- 第一人称 : 称呼自己 (I / we)
- 第二人称 : 称呼对方 (you)
- 第三人称 : 称呼自己和对方之外

- 第一人称单数 I; 复数 we
- 第二人称单数 you; 复数 you
- 第三人称单数 she / he / it; 复数 they

## 09 动词形态的用法区分 1

### 第一/二人称

- 当主语是第一/二人称时，动词的形态不会变化

```markdown
I play the guitar.
You play the guitar.
```

### 第三人称

- 当主语是第一/二人称单数时，动词之后要加 `s`

```markdown
He plays the guitar.
```

## 10 容易犯错的三单现

- 动词词尾为 o / s / x / ch / sh 时加 es

```markdown
go -> goes
```

- 词尾 a / e / i / o / u 以外的字母 + y 时则把 y 变成 i 再加 es

```markdown
study -> studies
```

## 11 动词形态的用法区分 2

- 一般动词只有在主语是第三人称单数时，才会用 “加 s 的形态”，其他情况就用 "原型"

```markdown
Mike plays the guitar.

Mike and Jack play the guitar.
```
