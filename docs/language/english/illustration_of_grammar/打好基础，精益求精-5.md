## 从句是什么

- 一个句子里面只能有一组"主语+动词"，但是 `that` 可以把多个"主语+动词"整合到一个句子里面
- `I think` 称为主句，`English is easy` 是 `think` 的宾语，称为从句

```markdown
I think that English is easy.
```

- 从句可以分为名词性从句、形容词性从句和状语从句等

```markdown
I think that English is easy. // 名词性从句
This is the book that she wrote. // 形容词性从句
It's raining when I got up. // 状语从句
```
