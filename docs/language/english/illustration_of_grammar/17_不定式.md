# 不定式

## 67 "不定式"是什么

- `to + 动词` 的形式称为不定式

- 使用不定式可以在整个句子的主要动词 (go) 以外，再增加一个动词 (study)，构成较复杂的句子

```markdown
I go to the library to study.
```

- 不管主语和时态是什么，不定式中的动词都使用原形

```markdown
He visits Beijing to see his sister.
I visited Beijing to see my sister.
```

- `to + 动词` 表示 "为了..."

```markdown
I go to the library to study. 我为了学习去图书馆
```

- `to + 动词` 脱离了动词原本的作用，主语和时态不能决定这个动词的形式，因此称其为 "不定式"

## 68 "为了..." "...的原因"

- 不定式可以作为副词 (称为副词性的不定式)，用来说明这是引起某种情感的原因，例如 `to see` 是引起 `I am glad` 的原因

```markdown
I am glad to see you. 我见到你很高兴
```

## 69 "需要...的" "可以...的"

- 不定式可以表示 "需要...的"

```markdown
I have a lot of homework to do. 我有很多作业需要做
```

- 不定式可以表示 "可以...的"

```markdown
I don't have time to watch TV. 我没有时间可以看电视
```

- `something to` 表示 "可以...的东西"

```markdown
I have something to eat. 我没有可以吃的东西
```

## 70 "做...事"

- 不定式表示 "做...事"。主要是作动词的宾语，也可能作句子的主语或补语。因为具有名词的功能，所以称为 "名词性的不定式"

```markdown
I like to talk with my friends.
I want to play basketball.
```
