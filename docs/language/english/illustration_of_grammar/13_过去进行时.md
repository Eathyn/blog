# 过去进行时

## 51 过去进行时是什么

- 过去正在进行的事情

```markdown
She was watching TV. 她当时正在看电视
```

- 过去进行时经常会和 `when` / `while` 等一起使用

```markdown
It was raining when I got up. 我起床的时候正在下雨
```

## 52 过去进行时的否定句和疑问句

- be 动词后加 `not` 形成否定句

```markdown
I was studying in the library. -> I was not studying in the library.
```

- be 动词放句首形成疑问句

```markdown
Were you studying in the library. -> Yes, I was. / No, I wasn't.
```

- 使用 `what` 等形成疑问句

```markdown
What were you doing? 你刚才在干嘛?
```

- 过去进行时表示的时间范围较窄，与中文的 “在做...” 不完全相同。例如汉语 “你昨天在干嘛?” 用英语表示为 `What did you do yesterday` 而不是 `What were you doing yesterday?`
