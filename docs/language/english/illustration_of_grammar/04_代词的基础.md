# 代词的基础

## 13 他 我们 等

- 代词是替代具体名词的词语

```markdown
"my brother" 为名词
"He" 为替代 "my brother" 的代词

This is my brother. He is in Beijing.
```

- 代词可以避免一再重复同样的话语

- I, you, he, she, it, we, they 被称为人称代词，充当句子的主语时称为主格

- this, that, these, those 直接用来指出人或物品，所以称为指示代词

## 14 他的 我们的 等

- 改变代词形态，表示 "某人的..." 的形态称为所有格

```markdown
my name -> "my" 为所有格
```

- 所有格只能用在名词之前

- 使用所有格时，名词之前不能加 `a` / `the`

```markdown
This is my bike. -> true
This is a my bike -> false
```

## 15 增加修饰的词 1

- 修饰名词的词为形容词

```markdown
a big apple -> big 为形容词
```

- 形容词也可以放在 be 动词之后

```markdown
This is interesting. -> 形容词为 interesting
```

- 修饰后面名词的形容词称为性质形容词或定语形容词；放在在 be 动词之后用来说明主语的形容词称为叙述形容词或表语形容词

```markdown
This is a big apple. -> big 为性质形容词或定语形容词
This is interesting. -> interesting 为叙述形容词或表语形容词
```

## 16 增加修饰的词 2

- 为句子增加修饰的词称为副词

```markdown
I am busy today. -> today 为副词
I am busy now. -> now 为副词
```

- 修饰名词的词为形容词；修饰动词或形容词的词称为副词

- `always` / `usually` / `often` / `sometimes` 等表示频率的副词在含一般动词的句子中通常放在动词前面，但在含 be 动词的句子中通常放在 be 动词后面

```markdown
I always walk to school. -> always 放在动词前
I am always angry. -> always 放在 be 动词之后
```

## 17 增加修饰的词 3

- 另一种修饰词称为介词短语

```markdown
I play guitar in the park. -> in the park 为介词短语; in 为介词
```
