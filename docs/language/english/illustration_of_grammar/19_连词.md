# 连词

## 72 连词 that

- `that` 两部分连接起来，通常可以省略

```markdown
I think that English is easy.
```

## 73 连词 when

```markdown
It was raining when I got up.
```

- 也可以把 `when...` 部分先说出来，后面要加逗号

```markdown
When I got up, it was raining.
```

## 74 连词 if, because

```markdown
I will help you if you are busy.
```

- `if... 可以提前`

```markdown
If you are busy, I will help you.
```

- 相对于主句，`when...`, `if...` 称为从句。因为在句子中作状语，因此称为状语从句
- 在状语从句中，即使是未来的事情，也要用一般现在时表示

```markdown
I will stay at home if it rains tomorrow.
```

## 复习测验

- 在讨论 "我以为...", "我当时就知道..." 等过去的事情时，必须注意用词的形态。`that` 后面的动词，原则上和前面的动词一样

```markdown
I thought that my mother is busy.
```
