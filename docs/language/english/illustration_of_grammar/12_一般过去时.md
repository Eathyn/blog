# 一般过去时

## 42 一般过去时是什么

- 大部分动词 + `ed` 后形成过去式

```markdown
I play baseball. 我(平常)打棒球
I played baseball yesterday. 我昨天打棒球
```

- 与一般现在时不同，一般过去时中的动词不会随着主语的不同而发生变化

```markdown
I play baseball. -> I played baseball.
He plays baseball. -> He played baseball.
```

## 43 要特别注意的过去式

- 一般动词后加 `ed`

```markdown
play -> played
```

- 以 `e` 结尾的词，加 `d`

```markdown
live -> lived
like -> liked
use -> used
```

- 以 `a, e, i, o, u` 以外的字母 + `y` 结尾的词，改 `y` 为 `i` 再加 `ed`

```markdown
study -> studied
carry -> carried
cry -> cried
try -> tried
```

- 双写最后一个字母，再加 `ed`

```markdown
stop -> stopped
```

- 不规则变化

```markdown
go -> went
come -> came
have -> had
get -> got
see -> saw
make -> made
read -> read
write -> wrote
```

## 44 一般过去时的否定句

- 一般过去时的否定句是在动词前面加上 `did not` / `didn't`，并把动词改为原型

```markdown
I went to school yesterday. -> I didn't go to school yesterday.
```

- 一般现在时的否定句，随着主语的不同，会有 `do not` 和 `does not` 的区分，但是一般过去时没有这种区分

```markdown
I don't go to school. -> I didn't go to school.
He doesn't go to school. -> He didn't go to school.
```

- `didn't` 中的 `did` 是助动词，因此后面使用动词原型

## 45 一般过去时的疑问句

- 询问过去是否做了某事用 `Did` 开头，回答用 `Yes, ... did.` / `No, ... did not`

```markdown
Did you watch TV last night? -> Yes, I did. / No, I didn't.
```

- 一般现在时的疑问句会随着主语的不同使用 `do` 和 `does`，但一般过去时的疑问句都是用 `did`

```markdown
Do you have a cat? -> Did you have a cat?
Does he have a cat? -> Did he have a cat?
```

## 46 "做了什么"

- 询问过去的事情，使用 `What` / `When` / ... + `did ...`

```markdown
What did you do last night?
When did you see him?
Where did she go?
How did you get this?
```

- 疑问句做主语时，不使用 `did`，以疑问句 + 动词过去式的语序表达

```markdown
What happened?
Who made this?
```

## 47 was 和 were

- 表示过去的状态使用 `was` / `were`

```markdown
He was hungry.
You were tired last night.
```

- 主语是 `I` 或第三人称单数( he, she, it )的时候，be 动词用 `was`；主语是 `you` 或复数的时候，be 动词用 `were`

```markdown
I was busy yesterday.
You were busy yesterday.
```

- 询问某物怎么样时，使用 `How was ...?` 来提问，通常用 `It was ...` 来回答

```markdown
How was the book? -> It was interesting.
```
