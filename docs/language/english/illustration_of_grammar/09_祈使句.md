# 祈使句

## 35 要求对方做某事

- 祈使句不需要主语

```markdown
Stand up.
```

- 增加 please 可以缓和命令的口吻

```markdown
Please stand up.
Stand up, please.
```

- 祈使句可以用来表示邀请或提议

```markdown
Have some tea.
```

- 指引方向时不需要加 please

```markdown
Go straight.
```

## 36 不要做... / 让我们做...吧

- 表达不可以做...，只需要在祈使句前增加 Don't
- 增加 please 可以缓和命令的口吻

```markdown
Open the door. -> Don't open the door.
```

- 使用 Let's 邀请别人一起做某事

```markdown
Let's sing together.
```
