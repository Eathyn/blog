# 复数形式

## 32 复数形式是什么

- 数量是一个时，一般需要在名词前加 `a`。如果名词的开头发元音，则加 `an`

```markdown
a cat
an apple
```

- 数量是两个或两个以上时，名词后面需要加 `s`，这种形态称为复数

```markdown
two cats
```

## 33 特殊的复数形式

### +es

- s, x, ch, sh 结尾的词加 es

```markdown
class -> classes
box -> boxes
```

### +ies

- 词尾为 a, e, i, o, u 之外的字母 + y 结尾的词，将 y 改为 i 再加 es

```markdown
country -> counties
city -> cities
```

### 不规则变化

```markdown
man -> men
woman -> women
child -> children
```

### 不可数名词

- 有些名词是不可数的，没有复数形式，例如 water

### How many

- How 可以结合其他词语做疑问句

```markdown
How many cats do you have?
How old are you?
```
