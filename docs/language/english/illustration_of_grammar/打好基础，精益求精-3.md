# 打好基础，精益求精 - 3

## in

### 在...里/内

```markdown
in the box
in the kitchen
in China
```

### 在某年/月/季节

```markdown
in 2012
in June
in winter
```

## on

### 在...上面/表面上

```markdown
on the table
on the wall
```

### 在...日期/星期

```markdown
on May 5
on Monday
```

## at

### 在某个地点上

```markdown
at school
at the bus stop
```

### 在某个时刻

```markdown
at six
at 10:30
```

## before 在...之前

```markdown
before dinner
```

## after 在...之后

```markdown
after school
```

## from 来自...

```markdown
a letter from him
```

## to 往...

```markdown
go to school
```

## with 和...一起

```markdown
go with him
```

## for

### 为了...

```markdown
buy a present for him
```

### 对于...而言

```markdown
good for you
```

### 表达时间长短

```markdown
walk for ten minutes
stay there for two weeks
```
